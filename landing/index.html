Como funciona?

Tudo automático, do PDF oficial do SESC até a mensagem no seu celular.

📄
Passo 1

Detecta o PDF

Todo dia verifica se o SESC publicou um novo PDF de programação no site oficial.

🤖
Passo 2

IA Extrai Tudo

O Gemini lê o PDF completo e extrai 100% dos eventos: shows, teatro, esportes, oficinas e mais.

🗄️
Passo 3

Salva no Banco

Eventos são persistidos em banco de dados local, sem duplicatas e com histórico completo.

📱
Passo 4

Alerta no Telegram

Você recebe mensagens formatadas e organizadas, sem precisar procurar nada no site.

Funcionalidades

O que o bot faz?

Pensado para quem quer aproveitar a cultura sem perder tempo procurando informação.

🧠

Extração com IA

O Gemini interpreta PDFs complexos e estrutura as informações: nome, data, local, horário, preço e classificação etária.

🔔

Alertas Inteligentes

Notificações diárias com o que acontece hoje, e semanais com a grade da semana — tudo direto no Telegram, sem abrir o site.

🚫

Zero Spam

Sistema de deduplicação garante que o mesmo evento nunca é enviado duas vezes. Banco de dados histórico rastreia tudo.

💾

Cache Inteligente

O PDF é baixado localmente e analisado só quando há novidade. Economia total de processamento e de tokens de IA.

🏷️

Emojis Contextuais

👶 infantil, 👴 terceira idade, 🎭 teatro, ⚽ esportes, 🎬 cinema — cada atividade tem seu emoji automático.

⚙️

100% Automatizado

Roda via cron do sistema operacional. Nenhuma interação humana necessária após configuração. Funciona 24/7.

Tipos de Notificação

3 tipos de alerta

Cada alerta tem um propósito diferente — nenhum dado é processado do Gemini nos disparos do dia a dia.

📥
Sync · Diário

Verificação do PDF

Todo dia verifica se há novo PDF. Se houver, baixa, lê com o Gemini e atualiza o banco de dados. Se não houver, não gasta nada.

node agenda.js sync

Daily · Diário
